webserv

WebServ is a high-performance, event-driven HTTP server built from scratch in C++ to mimic the functionality of Nginx while following RFC 9110 (HTTP/1.1) standards. It is designed to handle multiple virtual hosts, process HTTP requests efficiently, and support dynamic content execution via CGI.

This project is an implementation of a non-blocking, asynchronous web server that uses epoll for efficient I/O multiplexing. It supports routing, redirections, static file serving, directory listing, and request parsing with strict RFC compliance.
